https://thierrycohnen.github.io/YTvideosPlayer/ (Projet en cours de finition)
   est un programme de lecture par morceaux d'une vidéo Youtube. La possibilité de déplacer le curseur permet de se positionner dans la vidéo à l'endroit désiré. On peut ensuite appuyer sur le bouton 'Add Start' qui enregistre et affiche cette position dans une table HTML. Le bouton 'Add Stop' permet de définir la fin de l'intervalle de lecture.

Grâce au bouton '+', on choisit alors le nombre de répétitions de cet interval et finalement le 'Add X times' ajoute ces paramètres à la liste HTML.
Il est ainsi possible de créer plusieurs intervals et répétitions aux endroits choisis. Enfin, un click sur le 'Play All' permet la lecture suivie de tous ces intervals présents dans la table HTML affichée.

Code source disponible sur: https://github.com/ThierryCohnen/YTvideosPlayer
